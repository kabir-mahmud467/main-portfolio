# kabirmahmud.xyz Platform

Production-ready Express, MongoDB, EJS, and Tailwind foundation for a personal brand platform.

## Setup

```bash
cp .env.example .env
npm install
npm run build
npm run dev
```

Open `http://localhost:3000`.

## Architecture

- ES Modules
- Express MVC
- MongoDB with Mongoose
- EJS layouts and reusable partials
- Tailwind CSS
- Helmet, compression, sessions, rate limiting, flash messages, validation
